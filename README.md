# Professional Portfolio Verification Platform

This project is a professional portfolio platform where professionals can showcase their work, credentials, education, and verified background information.

## Purpose

- Help individuals create a digital portfolio for careers, freelance work, or personal branding.
- Allow employers to verify experience, certificates, education, and professional background from a trusted profile.
- Provide verification workflows for email, LinkedIn, GitHub, education records, identity documents, and background checks.
- Surface verified badges and employer trust scores that make it easier for hiring teams to evaluate candidates.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
